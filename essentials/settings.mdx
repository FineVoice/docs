## **Limitations and Benefits**

FineVoice API charges are based on the number of characters processed:

|              | **Character Count** | **Total Price** | **Concurrent Requests** | **Custom Voice Models** |
| :----------- | :------------------ | :-------------- | :---------------------- | :---------------------- |
| **Creator**  | 6,000,000           | \$90.00         | 5                       | 500                     |
| **Pro**      | 9,000,000           | \$135.00        | 5                       | 500                     |
| **Scale**    | 12,000,000          | \$156.00        | 10                      | 1000                    |
| **Business** | 15,000,000          | \$195.00        | 10                      | 1000                    |

#### **Notes:**

1. **Total Price** is the package price.
2. **Validity**: All packages are valid for 1 year.
3. **Concurrent Requests**: Number of tasks that can run simultaneously.
4. **Custom Voice Models**: The maximum number of voice models that can be created.
