## **Welcome**

You can generate a new API key at [**https://finevoice.ai/usercenter**](https://finevoice.ai/usercenter).

## **Quick Start**

See our [**Quick Start**](https://api.finevoice.ai/quickstart) guide to generate audio in under 5 minutes.

## [**​**](https://fvdocs.finevoice.ai/api-reference/introduction#create-a-voice-clone)**Create a Voice Clone**

Use our [**/trainmodel**](https://api.finevoice.ai/api-reference/endpoint/trainmodel) to create a voice clone model.

## [**​**](https://fvdocs.finevoice.ai/api-reference/introduction#generate-speech)**Generate Speech**

Use our [**/fsmstexttospeech**](https://api.finevoice.ai/api-reference/endpoint/changeprogress) to generate speech.
